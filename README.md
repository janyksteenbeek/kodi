# Kodi

**The Agentic IDE** — A native macOS Git client with integrated terminals for AI-powered development.

> Work in progress. Active development happening here :)

## Features

- **Git Management** — View diffs (unified & side-by-side), stage/unstage files, commit, push & pull
- **Integrated Terminals** — Multiple terminal sessions per repository with split panel support (bottom/right)
- **Quick Launch** — One-click launch for Terminal, Claude Code, Codex, or custom tools
- **Multi-Repository** — Work with multiple repos simultaneously via tab bar
- **Agentic Workflow** — Built for developers who use AI coding assistants as part of their daily workflow
